# space-invaders
my game of space invaders has five extenions from the seed prompts which are making it so that the highscore is saved between games and is displayed at the top right below the score of the current game. A oval UFO that goes across the top of the screen that when shot gives 30 points. That the game speeds up after 22 aliens are destroyed. a different type of alien that is a different color and shape that shoots at the player. The last change I made was to improve the visual style of the game by making the aliens distict from each other. The Game controller controlls the game loop and the movement of the game the game view controlls the visual style of the game. and the Game model controlls the hit detection, moving the aliens forward, and the score of the game.
